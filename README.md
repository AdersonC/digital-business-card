# Digital Business Card

Projeto de cartão de visita digital desenvolvido para praticar conceitos básicos de desenvolvimento web.

## Tecnologias utilizadas

- HTML5
- CSS3

## Funcionalidades

- Exibição de informações pessoais
- Layout estilizado
- Estrutura simples de página web

## Objetivo do projeto

Praticar a criação de páginas web e organização de elementos com HTML e CSS.

## Como executar

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador
